# TEDxNITW Website

The official website for TEDxNITW - the independently organized TED event at NIT Warangal.

Built from scratch with vanilla HTML, CSS, and JavaScript. No frameworks, no bloat - just clean code.

## What's Here

```
index.html      → Home page with the hero zoom effect + speaker reel
about.html      → About the event and venue blueprint
speakers.html   → Speaker archives (2017, 2018, 2020, 2021)
team.html       → The organizers
contact.html    → Contact info
```

## Running It

Just open `index.html` in your browser. That's it.

Or use Live Server if you want hot reload:
```bash
npx live-server
```

## The Archives

The speaker archives system supports multiple event years:
- **VOL I** - 2017 "Breaking the Mould"
- **VOL II** - 2018 
- **VOL III** - 2020 "Into the Matryoshka"
- **VOL IV** - 2021 TEDxNITWStudio

Click the timeline on the right to switch between volumes.

## Assets
